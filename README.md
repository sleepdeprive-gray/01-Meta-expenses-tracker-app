# Second Commit

<h1><center>Mini Project</center></h1>
## Introduction
This is a mini project that demonstrates the use of Git for version control. The project consists of a simple README file that is updated with each commit to show the progress of the project.

## Project Structure
The project structure is as follows:
```Mini-Project/
├── README.md
```


## Conclusion
This mini project serves as testing for basic fundamentals of React and Laravel. It is a simple web application that allows users to create, read, update, and delete (CRUD) tasks. The project is built using React for the frontend, Laravel for the backend and MySQL for database. It demonstrates the use of Git for version control and shows the progress of the project through commits.
