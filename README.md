<h3>Project Description</h3>
<p>
    This is a simple expenses tracker application built using Laravel for the backend and React for the frontend. It allows users to manage their expenses by adding, editing, and deleting expense records. The application also provides a summary of total expenses and categorizes them for better organization.
</p>
    
<h3>Features</h3>
<ul>
    <li>Expense Management: Users can add, edit, and delete expenses with details such as amount, category, and date.</li>
    <li>Expense Summary: The application provides a summary of total expenses in date range for better insights.</li>
    <li>Preloader: A preloader is displayed while the application is loading to enhance user experience.</li>
    <li>MySQL Database: The application uses MySQL to store expense data securely and efficiently.</li>
</ul>
